## Places to view the vault

The primary obsidian vault is located in the quartz repository. Each edit is pushed to Github, and then Quartz is generating static html files in Github pages that can be accessed over the internet. It is the most updated version of my vault. The second is a copy of the locally generated static html files with `npx quartz build` and then to the diary subdomain of my saiht.de site.

- Primary: [kreier.github.io/quartz](https://kreier.github.io/quartz/)
- Secondary: [diary.saiht.de](https://diary.saiht.de/)

## Vault for redundancy and statistics

While the primary vault exist in the quartz repository, this repository get's a copy from time to time. In order to have the full commit history inside this repository I created a separate branch `obsidian` that contains all commits of myself to the quartz vault.
